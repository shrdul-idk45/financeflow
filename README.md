FinanceFlow is a full-stack personal finance management web application that allows users to track income and expenses, view financial summaries, and analyze spending patterns through an intuitive dashboard.

Overview

FinanceFlow helps users:
-Record income and expenses.
-Categorize transactions.
-View balances and summaries.
-Analyze monthly and category-wise spending.
-Access data securely with authentication.
-The application uses Supabase for authentication and is deployed on Vercel using environment-based configuration.

Features

Authentication:

-Secure email and password authentication using Supabase.
-Persistent login sessions across page refreshes.

Dashboard:

-Displays total balance, total income, and total expenses.
-Provides a high-level financial overview.

Transactions:

-Add income and expense entries.
-Categorized and date-based records.
-Clear listing of transaction history.

Analytics:

-Category-wise expense distribution.
-Monthly spending trends.
-Visual representation of financial data.

User Interface:

-Clean and modern UI designed from Figma.
-Responsive layout.
-Light and dark theme support.

Deployment

-Production deployment on Vercel.
-Environment variables managed securely outside the repository.
